## About

I'm a senior cloud infrastructure engineer focused on the intersection of **DevSecOps** and **MLOps** — securing CI/CD pipelines, building Kubernetes-native platforms, and making machine learning workloads production-ready at scale.

At **Teradata**, I work as a CloudOps Specialist leading cloud infrastructure operations on GCP. Outside work, I run [**syssignals.com**](https://syssignals.com) — a technical education platform where every article ships real, working code. No hand-waving. No screenshots-only tutorials.

- 🏢 CloudOps Specialist @ [Teradata](https://www.teradata.com)
- ✍️ Writing the **30 Days of DevOps** series at [syssignals.com](https://syssignals.com)
- 🎯 Currently focused on: Kubernetes security, MLOps pipelines, GitOps
- 📍 India
